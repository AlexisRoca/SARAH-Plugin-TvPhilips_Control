Le plugin TvPhilips_Control permet de controler totalement sa télévision Philips

L'ip de la télévision est à placé dans le fichier .propTvPhilips_Control.prop :

{
  "modules": { 
    "TvPhilips_Control": {
      "description": "Controle des televisions Philips",
      "version": "1.0",
      "ip": "192.168.x.x"
    }
  }
}

Explication du pluggin:

J'ai ajouté un mode Télévision pour controler la télé sans à avoir à dire SARAH avant chaque commande Attention à bien articuler le premier mot de la commande

Pour l'activer il faut dire TvPhilips_Control.xml :

  • Sarah, active le mode television

Et pour en sortirTvPhilips_Control.xml :

  • Merci ce sera tout Sarah


Dans le fichier XML il faut ajouter les appareils qui sont branché à votre télévision ainsi que leur port sur lequel il sont branchés

Exemple:

<item>affiche la x box trois cent soixante<tag>out.action.command="hdmi1"</tag></item>
<item>met la x box trois cent soixante<tag>out.action.command="hdmi1"</tag></item>
<item>remet la x box trois cent soixante<tag>out.action.command="hdmi1"</tag></item>
<item>affiche la x box ouane<tag>out.action.command="hdmi2"</tag></item>
<item>met la x box ouane<tag>out.action.command="hdmi2"</tag></item>
<item>remet la x box ouane<tag>out.action.command="hdmi2"</tag></item>
<item>affiche la yoga<tag>out.action.command="hdmi3"</tag></item>
<item>met la yoga<tag>out.action.command="hdmi3"</tag></item> 
<item>remet la yoga<tag>out.action.command="hdmi3"</tag></item> 
<item>affiche la télé<tag>out.action.command="tv"</tag></item>
<item>met la télé<tag>out.action.command="tv"</tag></item> 
<item>remet la télé<tag>out.action.command="tv"</tag></item>

on place son appareil dans la phrase à dire et son port dans la variable commande

Pour les intéressé, l'api des télévision philips se trouve sur à l'adresse de votre télévision sur le port 1925 (exemple: 192.168.0.1:1925)
